# Projeto desenvolvido durante a NLW/Together

## Evento da Rocketseat na trilha de React.js, utilizando também o Firebase para armazenamento. O objetivo do evento foi criar uma plataforma para filtragem de perguntas durante uma live, desenvolvendo um ranking das mesmas. Tornando assim a aplicação um facilitador para os apresentadores durante suas lives.
